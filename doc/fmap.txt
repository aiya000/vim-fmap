INTRODUCTION					*fmap-introduction*

*fmap* takes a way for mappings of |f|, that is like |nnoremap|, but for vim's
|f|.

==============================================================================
USAGE						*fmap-usage*

If you install this plugin, and you add the following line into your .vimrc,
>
	autocmd! VimEnter * FNoreMap pb (
	autocmd! VimEnter * FNoreMap pe )

	nmap <leader>f <Plug>(fmap-forward-f)
<
Now you can shot '(' and ')' with <leader>fpb and <leader>fpe !

==============================================================================
TERM						*fmap-term*

{f-mapping}					*fmap-f-mapping*
	The key mapping for |f|. This is exactly the plugin's purpose!

g:fmap_mappings					*g:fmap_mappings*
	default value: copy(g:fmap_default_mappings)

	For the user preference.
	You can define this like
>
	let g:fmap_mappings = [
	    \ { 'strokes': ['T'], 'target': '‚Ä¶'},
	    \ { "strokes": ["p"], "target": "(" },
	    \ { "strokes": ["P"], "target": ")" },
	]
<
	This priority is higher than |g:fmap_default_mappings|
	(|g:fmap_default_mappings| is ignored).

	You would define it like below, if you want to avoid it.
>
	let g:fmap_mappings = extend(g:fmap_default_mappings, [
	    \ { 'strokes': ['T'], 'target': '‚Ä¶'},
	    \ { "strokes": ["p"], "target": "(" },
	    \ { "strokes": ["P"], "target": ")" },
	\ ])
<
	Also |g:fmap_default_mappings| is used simply if this is not defined.

g:fmap_use_default_keymappings		*g:fmap_use_default_keymappings*
	default value: v:true

g:fmap_escape_keys			*g:fmap_escape_keys*

------------------------------------------------------------------------------
COMMANDS					*fmap-commands*

:FNoreMap {stroke} {char}				*:FNoreMap*
	Adds the {stroke} and the {char} as an {f-mapping}, to
	|g:fmap_mappings|, if the {char} is not existent as an element of
	g:fmap_mappings.target .

	Or adds the {stroke} to g:fmap_mappings.strokes if
	"g:fmap_mappings.target ==# {char}" satisfies.

	e.g.
	This makes {"strokes": ["."], "target": "„ÄÇ"} into |g:fmap_mappings|.
>
	FNoreMap . „ÄÇ
<
	Also this makes {"strokes": [",", "x"], "target": "„ÄÅ"} into
	|g:fmap_mappings|.
>
	FNoreMap , „ÄÅ
	FNoreMap x „ÄÅ
<

KEY-MAPPINGS					*fmap-key-mappings*

<Plug>(fmap-forward-f)				*<Plug>(fmap-forward-f)*
	available mode: normal
	default mappings (global):
		normal:	<leader>f

	Shot forward {char} with a {keys} like |f|.

<Plug>(fmap-backward-f)				*<Plug>(fmap-backward-f)*
	available mode: normal
	default mappings (global):
		normal:	<leader>F

	Shot forward {char} with a {keys} like |F|.

<Plug>(fmap-forward-t)				*<Plug>(fmap-forward-t)*
	available mode: normal
	default mappings (global):
		normal:	<leader>t

	Shot forward {char} with a {keys} like |t|.

<Plug>(fmap-backward-t)				*<Plug>(fmap-backward-t)*
	available mode: normal
	default mappings (global):
		normal:	<leader>T

	Shot forward {char} with a {keys} like |T|.

						*fmap-default-key-mappings*

normal mode

<leader>F	<Plug>(fmap-backward-f)
